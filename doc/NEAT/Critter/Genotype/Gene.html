Gene Specification

The Gene specifices a singlular input and output neuron, which represents a connection between them, along with the weight of that connection, which may be positive, negative, or zero.

def self.[](genotype, input, output, weight = 0.0, innov = nil)
  g = Gene.new genotype
  g.in_neuron = (input.kind_of? Symbol) ? input : input.name
  g.out_neuron = (output.kind_of? Symbol) ? output : output.name
  g.weight = weight
  g.innovation = innov unless innov.nil?
  return g
end
